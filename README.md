# Triplo AI Updates

![Triplo AI Logo](https://media.swipepages.com/2023/1/63d5e3e624ea3f00106d0797/logo_34a853.svg "Triplo AI Logo")

Triplo’s public releases live here: the desktop installer, the unified Docker image, and supporting web assets.

## Download

- Desktop users can grab the latest signed installer from the **Releases** tab or directly from [triplo.ai](https://triplo.ai).
- Existing installations auto-update, so manual downloads are only needed for fresh installs or offline setups.

## Contents

- `Releases · Elbruz-Technologies-triplo.url` – shortcut to the official release feed.
- `c/scripts/triplo-ai` – Docker/web UI workspace used to build and document the unified container.
